## Overview
This project is a practical implementation from the **Data Structures course** and represents **Level 1 of Data Structures applications** within **Level 13 of the Programming Roadmap** designed by Dr. Mohammed Abu-Hadhoud.

The goal of this project is to build fundamental data structures from scratch using **C++**, focusing on understanding how these structures work internally rather than relying on built-in libraries.

---

## Project Idea
The project demonstrates how complex data structures can be built step by step starting from a foundational structure.

1. **Doubly Linked List**
   - Implemented from scratch as a reusable class.
   - Supports insertion, deletion, traversal, and search.

2. **Queue Implementation**
   - Built using the Doubly Linked List class.
   - Demonstrates the **FIFO (First In, First Out)** concept.

3. **Stack Implementation**
   - Built on top of the existing structure.
   - Demonstrates the **LIFO (Last In, First Out)** concept.

This layered design highlights the importance of **abstraction, modular design, and code reusability**.
